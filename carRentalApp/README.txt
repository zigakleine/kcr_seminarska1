Navodila za poganjanje in buildanje aplikacije:

- premaknemo se v mapo
- poženemo ukaz npm install, ki naloži javascript knižnjice
- če hočemo aplikacijo samo pognati, poženemo ukaz npm run start

- če hočemo zbuildati aplikacijo, poženemo ukaz npm run package-mac, npm run package-win, npm run package-linux, 
glede na operacijski sistem, ki ga uporabljamo.
